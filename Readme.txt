# My Personal Website

## Description

This is my personal website built with HTML, CSS, and PHP. It showcases my portfolio, projects, and provides information about me.

## Features

- About Me Section
- Portfolio Showcase
- Contact Form
- Responsive Design

## Technologies Used

- HTML
- CSS
- Javascript
- Bootstrap
- PHP

## Project Structure

- `/css`: Contains stylesheets.
- `/images`: Store your project images here.
- `/js`: JavaScript files.
- `/portfolio`: PHP files for contact form.
- `index.php`: Main homepage.
